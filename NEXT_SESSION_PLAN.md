# ПЛАН ПЕРЕДЕЛКИ В СТИЛЕ SHORT PAR 4

## Цель: Полная переделка A.V.K. SPORT в стиле Short Par 4 (subscription golf service)

## Ключевые элементы для реализации:

### 1. ЦВЕТОВАЯ СХЕМА
- Основной: Ярко-зеленый (#00B67A или похожий)
- Акцент: Черный (#000000)
- Фон: Белый (#FFFFFF)
- Дополнительный: Серый для текста

### 2. ГЛАВНАЯ СТРАНИЦА (HomePage)
- [ ] Hero с большой CTA кнопкой "Присоединиться"
- [ ] Бесконечная карусель брендов (Nike, Adidas, и т.д.)
- [ ] Секция "Как это работает" (4 шага)
- [ ] Membership карточки (2-3 тарифа)
- [ ] Past Shipments - примеры прошлых заказов
- [ ] FAQ секция
- [ ] Google Reviews integration (можно мокнуть)

### 3. КОМПОНЕНТЫ
- [ ] BrandCarousel - бесконечная прокрутка логотипов
- [ ] MembershipCard - карточки подписки с ценами
- [ ] HowItWorks - пошаговая инструкция
- [ ] TestimonialSlider - слайдер отзывов

### 4. ТИПОГРАФИКА
- Крупные, жирные заголовки
- Курсив для акцентов
- Просторные отступы

### 5. HEADER
- Минималистичный
- Логотип слева
- Навигация по центру
- Корзина справа

### 6. SUBSCRIPTION МОДЕЛЬ (опционально - можно адаптировать)
- Membership планы
- "Vault" - магазин со скидками

## Технические детали:
- Все изображения брендов можно использовать placeholder
- Цвета: зеленый + черный + белый
- Анимации: плавные transitions, карусели
- Мобильная адаптивность обязательна
